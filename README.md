__What is Bolt?__

Bolt is a modular terminal text editor writen in C# for UNIX based systems


#### Features in current build
- [x] Rich text editor inspired by GNU Nano
- [x] Configuration file (~/.bolt)
- [x] Modular design
- [ ] Shortcuts
- [ ] Syntax highlighting
- [ ] Plugin system and API


#### Supported platforms
- `OS X`: Yes
- `Linux`: Yes
- `Windows`: Untested
